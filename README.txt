# Library Management System

## Introduction
The Library Management System is designed to manage the operations of a library effectively. It handles tasks such as managing book inventories, user memberships, book checkouts, and returns. This system aims to simplify the process for both librarians and members, enhancing the overall user experience.

## Features

### For Librarians
- **Add New Books:** Enter details of new books into the system.
- **Update Book Status:** Change the status of books to reflect current availability.
- **Manage Members:** Register new members and manage existing member profiles.
- **Track Overdue Books:** View and manage a list of books that are overdue.

### For Members
- **Create Profiles:** Members can create and manage their profiles.
- **Check Out Books:** Browse and check out available books.
- **Return Books:** Return books by the due date.
- **Renew Memberships:** Members can renew their memberships annually.
